# 🦋 Invertebrados - Jogo Educacional

Jogo educacional interativo sobre animais invertebrados, desenvolvido com Next.js e Tailwind CSS.

## 🎮 Funcionalidades

- **8 grupos** de invertebrados (Insetos, Aracnídeos, Crustáceos, Moluscos, Anelídeos, Equinodermos, Poríferos, Cnidários)
- **15 animais** com fichas completas: curiosidades, habitat, alimentação e características
- **Quiz interativo** com 10 perguntas aleatórias, timer de 20 segundos e feedback explicativo
- **Sistema de pontos** e **conquistas (badges)**
- **Progresso salvo** durante a sessão

## 🚀 Deploy no Vercel

### Opção 1: Via Interface Web (mais fácil)

1. Faça upload do projeto para o GitHub
2. Acesse [vercel.com](https://vercel.com) e faça login
3. Clique em "Add New Project" → Import do GitHub
4. Selecione o repositório → clique "Deploy"
5. Pronto! O Vercel detecta Next.js automaticamente

### Opção 2: Via CLI

```bash
npm install -g vercel
vercel login
vercel
```

## 💻 Rodar Localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`

## 🛠️ Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
